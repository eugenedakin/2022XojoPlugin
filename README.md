# 2022XojoPlugin
Make a Xojo 2022 r1.1 plugin from scratch using Visual Studio 2022 in Windows

This is the code to make a plugin method which adds two numbers together. 
  - Compatible with Windows 11
  - Xojo IDE: 2022 r1.1
  - Xojo API: API 2
  - Level: Advanced
